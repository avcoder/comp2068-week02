Summary


  • JS has matured due to AJAX, Gmail/Google Maps, JQuery, Node, Chrome (V8), ES6,
  • The web has changed pretty quickly within the last 20 years. More and more logic moves from the server side to the client side. JS moved to the server (i.e. Node) The server part can be reduced to just serve static files and data over REST services.
  • More complex JavaScript needs to be written on the client side (i.e. Angular, React, Vue)
  • Web technology also moved to the desktop (i.e. Electron)
